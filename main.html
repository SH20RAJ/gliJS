gli.js

A lightweight JavaScript library for DOM manipulation, event handling, AJAX requests, and animations.

Getting Started

To use gli.js, include the script file in your HTML:

<script src="https://cdn.jsdelivr.net/gh/SH20RAJ/gliJS/gli.min.js"></script>
